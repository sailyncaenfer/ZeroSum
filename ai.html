How ZeroSum Works

Swipe (or use arrow keys / WASD) to slide every tile in one direction. Tiles glide until they hit the edge of the 4×4 board or another tile.

Tiles

5 Black tiles hold positive numbers. 5 White tiles hold negative numbers. Every tile is drawn from a shuffled deck of the values −10 through 10 (0 is never dealt on its own), so you'll always see a mix of both colors.

Cancelling

When a move pushes two tiles together, they interact only if they're exact opposites — equal magnitude, mismatched color, like 7 and 7. That pair cancels out completely and both tiles vanish from the board.

Combining

Small tiles are flexible: any 1, 2, or 3 (or its negative) can fuse with whatever tile it's pushed into, as long as the result doesn't exceed the current maximum tile value. The two tiles become one tile holding their sum. Any other pairing — same value, same color, or unrelated numbers above 3 — just blocks and stacks up without merging.

Scoring

Each cancellation scores 10× the tile's magnitude — cancelling a pair of 7s scores 70 points. A combine merge instead scores the magnitude of the smaller tile (1, 2, or 3 points).

Next Tiles

The column beside the board previews the next few tiles that will be dealt in. Tiles are drawn from a shuffled bag of every value up to the current maximum (starting at −10 to 10, excluding 0); once a bag runs out, a fresh one is shuffled. A dashed line in the preview marks the start of that next bag.

Raising the Ceiling

Every 10,000 points raises the maximum tile value by 1. You'll always finish the bag you're currently drawing from before any bigger tiles start appearing — the wider range only applies to bags shuffled after that.

Game Over

After every successful move, one new tile appears in a random empty cell. The game ends when the board is full and no move in any direction would allow a cancellation. Press R or tap Restart any time to start a fresh game.

AI Auto-Play

The AI button hands control to an expectimax search that looks ahead over its own possible moves and the tile that lands next. It searches iteratively -- depth 1, then 2, then 3, and so on, keeping the best move found so far and going one level deeper each time -- then plays the resulting move immediately.

The Depth field is a target, not a hard cap: if the search finishes that depth with time to spare before Min Time has elapsed, it keeps going deeper until that minimum is reached. Max Time is a hard ceiling -- the search always stops and plays its best move so far once that much time has passed, even mid-depth.

Replays

Tap Save Replay any time to download a compact .zsr replay file of the current game so far, including any moves played by the AI. Open it in replay.html to step or scroll through every move.
